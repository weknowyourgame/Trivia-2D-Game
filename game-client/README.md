# Phaser + React Starter

Minimal starter to build a Phaser game inside a React app using Vite.

Quick start:

1. Install dependencies:

```bash
cd phaser-react-starter
npm install
```

2. Run dev server:

```bash
npm run dev
```

Open `http://localhost:5173` and click the green square.

Notes:

- This starter uses Phaser 3 and React 18.
- No external assets included; add them under `public/` or load via `preload()`.
